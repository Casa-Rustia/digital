# Casa Rustia Creative Digital Services Website

Static website package for `casarustiadigital.com`.

## Latest cleanup pass

- Preserved the approved design, logo, colors, pages, prices, timelines, and form flow.
- Cleaned duplicate logo and navigation CSS rules that could cause inconsistent scaling.
- Tightened responsive navigation behavior for desktop, tablet, and mobile screens.
- Improved the mobile menu script so it closes on link click, outside click, Escape key, and desktop resize.
- Added safe image loading attributes where appropriate.
- Rechecked local page links, section anchors, image references, form actions, and ARIA references.

## Main files

- `index.html` — home page
- `contact.html` — free consultation form
- Service pages — dedicated inquiry pages for each service
- Package pages — dedicated inquiry pages for each offer
- `thank-you.html` — confirmation page after form submission
- `styles.css` — responsive styling
- `script.js` — sticky header and mobile menu behavior
- `404.html` — custom error page
- `CNAME` — custom domain setting
- `robots.txt` and `sitemap.xml` — SEO basics
- `assets/` — logos, icons, and package banner visuals

## Form delivery

Inquiry forms use FormSubmit and are configured to send to `casarustia@gmail.com`. The first live submission may require a one-time confirmation from that inbox before free delivery is activated.
